# Exercíco 59

Criar um programa que leia dois valores

Será exibido uma lista de opção
      [1] Somar
      
      [2] Multiplicar
      
      [3] Maior valor
      
      [4] Errei! Digitar valores correto
      
      [5] Sair

O programa deve realizar a operação solicitada quando escolher umas das opções acima!
